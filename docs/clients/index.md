# Para Clientes

Bem-vindo à documentação UniRadio para clientes! Este guia irá ajudá-lo a enviar exames e acessar seus laudos radiológicos.

## O Que Você Pode Fazer

Como cliente no UniRadio, você pode:

- 📤 **Enviar Exames** - Fazer upload de imagens e informações de exames para análise
- 🖼️ **Integração PACS** - Enviar exames diretamente do seu equipamento de imagem
- 📋 **Acessar Laudos** - Visualizar e baixar seus laudos radiológicos
- 👤 **Gerenciamento de Perfil** - Atualizar as informações da sua clínica
- ✏️ **Editar Exames** - Modificar detalhes dos exames antes da análise
