# Como Alterar Minha Senha

Aprenda como atualizar a senha da sua conta na plataforma UniRadio.

## Passo a Passo

### 1. Acesse a plataforma UniRadio

Navegue até [app.uniradio.vet.br/exams](https://app.uniradio.vet.br/exams)

### 2. Acesse o menu do usuário

Clique no seu nome ou ícone de usuário no canto superior direito para abrir o menu do usuário.

![Menu do usuário](/images/clients/security/password-change/01-menu-user.jpeg)

### 3. Clique em "Alterar Senha"

![Alterar Senha](/images/clients/security/password-change/02-change-password.jpeg)

### 4. Preencha o formulário

Informe a nova senha e a confirmação de senha.

![Formulário de alteração de senha](/images/clients/security/password-change/03-formulario.jpeg)

### 5. Clique em "Salvar" para confirmar

![Salvar nova senha](/images/clients/security/password-change/04-submit.jpeg)

## Precisa de Ajuda?

Confira nossas [Perguntas Frequentes](/faq) ou entre em contato com seu radiologista se encontrar problemas.
