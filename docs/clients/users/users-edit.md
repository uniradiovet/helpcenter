# Como Editar Usuários da Conta

Aprenda como acessar e editar os usuários vinculados à sua conta na plataforma UniRadio.

## Passo a Passo

### 1. Acesse a plataforma UniRadio

Navegue até [app.uniradio.vet.br/exams](https://app.uniradio.vet.br/exams)

### 2. Acesse o menu do usuário

Clique no seu nome ou ícone de usuário no canto superior direito para abrir o menu do usuário.

![Menu do usuário](/images/clients/users/users-edit/01-menu.jpeg)

### 3. Clique em "Usuários"

![Usuários](/images/clients/users/users-edit/02-users.jpeg)

### 4. Selecione o usuário que deseja editar

Clique sobre o usuário na lista.

![Selecionar usuário](/images/clients/users/users-edit/03-select-user.jpeg)

### 5. Edite as informações desejadas

Altere os campos que deseja atualizar, como nome, e-mail ou permissões.

![Editar campos do usuário](/images/clients/users/users-edit/04-edit-fields.jpeg)

### 6. Clique em "Salvar" para confirmar

![Salvar alterações](/images/clients/users/users-edit/05-submit.jpeg)

## Precisa de Ajuda?

Confira nossas [Perguntas Frequentes](/perguntas-frequentes) ou entre em contato com seu radiologista se encontrar problemas.
