# Como Editar um Exame

Aprenda editar os dados de um exame na plataforma UniRadio.

## Passo a Passo

### 1. Acesse a página de Exames

Navegue até [app.uniradio.vet.br/exams](https://app.uniradio.vet.br/exams)

### 2. Selecione o exame

Clique sobre o exame que deseja editar na lista.

![Selecionar exame](/images/clients/exam/exam-edit/01-select-exam.jpeg)

### 3. Clique em "Editar"

![Botão Editar](/images/clients/exam/exam-edit/02-edit.jpeg)

### 4. Edite as informações desejadas

Altere os campos necessários no formulário.

![Formulário de edição](/images/clients/exam/exam-edit/03-cranio.jpeg)

### 5. Clique em "Salvar" para salvar as alterações

![Submit](/images/clients/exam/exam-edit/04-submit.jpeg)

## Descrição dos campos

Para entender o significado de cada campo deste formulário, consulte [Campos da criação de exame](/clients/exam/exam-fields).

## Precisa de Ajuda?

Confira nossas [Perguntas Frequentes](/perguntas-frequentes) ou entre em contato com seu radiologista se encontrar problemas.
