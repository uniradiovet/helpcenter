# How to Filtrar Laudos

Aprenda como utilizar os filtros de busca para localizar laudos específicos na plataforma UniRadio.

::: warning Filtros por status
Os filtros são aplicados **dentro do status selecionado**. Ou seja, a busca não percorre todos os status simultaneamente — selecione primeiro o status desejado (ex.: "Finalizado") antes de aplicar os filtros.
:::

## Step by Step

### 1. Access the UniRadio platform

Navegue até [app.uniradio.vet.br/exams](https://app.uniradio.vet.br/exams)

### 2. Acesse a aba de laudos

Clique no status de "Finalizado" para visualizar a lista dos laudos.

![Lista de laudos](/images/en/clients/report/report-filter/01-lista-reports.jpeg)

### 3. Clique no botão "Filtrar"

![Filtrar](/images/en/clients/report/report-filter/02-filtrar.jpeg)

### 4. Preencha os campos de filtro desejados

Utilize um ou mais campos para refinar os resultados. Não é necessário preencher todos.
::: tip Campos disponíveis

- **Médico Solicitante** — nome do médico que solicitou o exame.
- **Tutor** — nome do responsável pelo animal.
- **Nome do Paciente** — nome do animal.
- **Palavras-chave** — termos presentes no laudo ou no exame.
- **Radiologista** — nome do radiologista responsável pelo laudo.
- **Espécie** — espécie do animal (ex.: Canina, Felina).
- **Suspeita clínica** — hipótese diagnóstica informada no envio do exame.
- **Região** — região anatômica examinada (ex.: Abdômen, Tórax).
- **Tipo** — modalidade do exame (ex.: Radiografia, Ultrassom).
- **Categoria do exame** — classificação atribuída ao exame.
- **De / Até (envio do exame)** — intervalo de datas em que o exame foi enviado à plataforma.
- **De / Até (envio do laudo)** — intervalo de datas em que o laudo foi emitido.
  :::

![Campos de filtro](/images/en/clients/report/report-filter/03-fields-filter.jpeg)

### 5. Clique em "Filtrar" para aplicar

![Aplicar filtro](/images/en/clients/report/report-filter/04-apply-filter.jpeg)
