# How to Import Client Accounts

Learn how to import client data from a CSV file in UniRadio by mapping fields correctly to ensure records are synced.

## Step by Step

### 1. Access the UniRadio platform

Sign in to your account and go to [app.uniradio.vet.br/exams](https://app.uniradio.vet.br/exams)

### 2. Open the Clients section

Click **"Clients"** in the navigation menu to open your client account list.

![Clients Section](/images/en/radiologists/clients/import-clients/01-clientes.jpeg)

### 3. Open the options menu

Click the options button (dropdown) to display available actions for the client list.

![Toggle Dropdown](/images/en/radiologists/clients/import-clients/02-dropdown.jpeg)

### 4. Click "Import Clients"

Select **"Import Clients"** in the menu to open the import panel.

![Import Clients](/images/en/radiologists/clients/import-clients/03-importar-clientes.jpeg)

### 5. Select the CSV file

Click **"Select"** and choose the CSV file with the client data you want to import.

![Select file](/images/en/radiologists/clients/import-clients/04-selecionar.jpeg)

### 6. Upload the file

Click **"Submit"** to upload the selected file.

![Submit](/images/en/radiologists/clients/import-clients/05-submit.jpeg)

### 7. Map the fields

Match CSV columns to the corresponding system fields. Make sure each column is mapped correctly for accurate imports.

![Map fields](/images/en/radiologists/clients/import-clients/06-mapear-campos.jpeg)

### 8. Click "Import"

Click **"Import"** to start the process. The import runs in the background and you will be notified when it finishes.

![Import](/images/en/radiologists/clients/import-clients/07-importar.jpeg)

To understand what each import mapping field means, see [Mapping Field Definitions](/en/radiologists/fields/fields-clients-import).

::: tip Tip
You can track previous imports in [Previous Imports](/en/radiologists/clients/import-clients-history).
:::
