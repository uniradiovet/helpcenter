# How to Reenviar Convite para Usuários em Massa

Aprenda como reenviar convites pendentes para múltiplos usuários de uma clínica de uma só vez na plataforma UniRadio.

## Step by Step

### 1. Access the UniRadio platform

Sign in to your account and go to [app.uniradio.vet.br/exams](https://app.uniradio.vet.br/exams)

### 2. Acesse a lista de clientes

No menu lateral, clique em **"Clientes"** para visualizar as clínicas cadastradas.

![Clientes](/images/en/radiologists/client-users/bulk-resend-invitation/01-clients.jpeg)

### 3. Selecione a clínica

Clique na clínica cujos usuários deseja gerenciar.

![Selecionar clínica](/images/en/radiologists/client-users/bulk-resend-invitation/02-select-clinic.jpeg)

### 4. Acesse os usuários da conta

Dentro da clínica, clique na aba de usuários para visualizar os usuários cadastrados.

![Usuários da conta](/images/en/radiologists/client-users/bulk-resend-invitation/03-users-account.jpeg)

### 5. Selecione os usuários desejados

Marque as caixas de seleção dos usuários para os quais deseja reenviar o convite.

![Selecionar usuários](/images/en/radiologists/client-users/bulk-resend-invitation/04-select-users.jpeg)

### 6. Clique no dropdown de ações

Com os usuários selecionados, clique no menu suspenso de ações em massa.

![Dropdown de ações](/images/en/radiologists/client-users/bulk-resend-invitation/05-dropdown.jpeg)

### 7. Selecione "Reenviar Convite Para Usuários Selecionados"

No menu, escolha a opção **"Reenviar Convite Para Usuários Selecionados"**.

![Reenviar convite](/images/en/radiologists/client-users/bulk-resend-invitation/06-resend-invite.jpeg)

### 8. Clique em "Executar Ação"

Confirme a operação clicando em **"Executar Ação"** para reenviar os convites aos usuários selecionados.

![Executar ação](/images/en/radiologists/client-users/bulk-resend-invitation/07-executar-acao.jpeg)
