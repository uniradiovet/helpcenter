# How to Get a User's Password Reset Link

Learn how to generate and copy the password reset link for a user linked to a client account on the UniRadio platform.

## Step by Step

### 1. Access the UniRadio platform

Sign in to your account and go to [app.uniradio.vet.br/exams](https://app.uniradio.vet.br/exams)

### 2. Access "Clients"

In the side menu, click on **"Clients"** to view the list of registered clients.

![Clients](/images/en/radiologists/client-users/user-password-reset-link/01-clients.jpeg)

### 3. Select the clinic

Click on the clinic to open its management page.

![Select clinic](/images/en/radiologists/client-users/user-password-reset-link/02-select-clinic.jpeg)

### 4. Access "Account Users"

Inside the client profile, click on **"Account Users"** to view the users linked to it.

![Account Users](/images/en/radiologists/client-users/user-password-reset-link/03-users-account.jpeg)

### 5. Select the user

Click on the name of the user for whom you want to generate the password reset link.

![Select user](/images/en/radiologists/client-users/user-password-reset-link/04-select-user.jpeg)

### 6. Click on "Actions"

On the user page, click the **"Actions"** button to display the available options.

![Actions](/images/en/radiologists/client-users/user-password-reset-link/05-acoes.jpeg)

### 7. Click on "Password Reset Link"

In the actions menu, select **"Password Reset Link"** to generate the reset link.

![Password Reset Link](/images/en/radiologists/client-users/user-password-reset-link/06-link-reset-password.jpeg)

### 8. Click "Copy Link"

Click **"Copy Link"** to copy the link and send it to the user.

![Copy Link](/images/en/radiologists/client-users/user-password-reset-link/07-copiar-link.jpeg)
