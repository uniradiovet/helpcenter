# Como Importar Contas de Clientes

Aprenda como importar dados de clientes a partir de um arquivo CSV na plataforma UniRadio, mapeando os campos para garantir que os registros sejam sincronizados corretamente.

## Passo a Passo

### 1. Acesse a plataforma UniRadio

Faça login em sua conta e navegue até [app.uniradio.vet.br/exams](https://app.uniradio.vet.br/exams)

### 2. Acesse a seção de Clientes

Clique em **"Clientes"** no menu de navegação para acessar a lista de contas dos seus clientes.

![Seção Clientes](/images/radiologists/clients/import-clients/01-clientes.jpeg)

### 3. Abra o menu de opções

Clique no botão de opções (dropdown) para exibir as ações disponíveis para a listagem de clientes.

![Toggle Dropdown](/images/radiologists/clients/import-clients/02-dropdown.jpeg)

### 4. Clique em "Importar Clientes"

Selecione a opção **"Importar Clientes"** no menu para abrir o painel de importação.

![Importar Clientes](/images/radiologists/clients/import-clients/03-importar-clientes.jpeg)

### 5. Selecione o arquivo CSV

Clique no campo **"Selecionar"** e escolha o arquivo CSV com os dados dos clientes que deseja importar.

![Selecionar arquivo](/images/radiologists/clients/import-clients/04-selecionar.jpeg)

### 6. Envie o arquivo

Clique em **"Submit"** para fazer o upload do arquivo selecionado.

![Submit](/images/radiologists/clients/import-clients/05-submit.jpeg)

### 7. Mapeie os campos

Relacione as colunas do seu arquivo CSV com os campos correspondentes do sistema. Certifique-se de que cada coluna esteja mapeada corretamente para garantir que os dados sejam importados com precisão.

![Mapear campos](/images/radiologists/clients/import-clients/06-mapear-campos.jpeg)

### 8. Clique em "Importar"

Clique no botão **"Importar"** para iniciar o processo. A importação será processada em segundo plano e você será notificado ao final.

![Importar](/images/radiologists/clients/import-clients/07-importar.jpeg)

Para entender o significado de cada campo deste mapeamento de importações, consulte [Campos do mapeamento](/radiologists/fields/fields-clients-import).

::: tip Dica
Você pode acompanhar o histórico de importações realizadas na seção [Importações anteriores](/radiologists/clients/import-clients-history)
:::
