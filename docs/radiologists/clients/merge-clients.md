# Como Mesclar Contas de Clientes Duplicadas

Aprenda como consolidar contas de clientes duplicadas na plataforma UniRadio, mantendo um único perfil com todos os dados unificados.

## Passo a Passo

### 1. Acesse a plataforma UniRadio

Faça login em sua conta e navegue até [app.uniradio.vet.br/exams](https://app.uniradio.vet.br/exams)

### 2. Acesse a lista de clientes

No menu lateral, clique em **"Clientes"** para visualizar todas as contas cadastradas.

![Lista de clientes](/images/radiologists/clients/merge-clients/01-clients.jpeg)

### 3. Selecione as contas duplicadas

Marque as caixas de seleção das contas que deseja mesclar.

:::warning
É necessário selecionar **pelo menos duas contas** para realizar a mesclagem.
:::

![Selecionar contas](/images/radiologists/clients/merge-clients/02-select-accounts.jpeg)

### 4. Clique no dropdown de ações

Com as contas selecionadas, clique no menu suspenso de ações em massa.

![Dropdown de ações](/images/radiologists/clients/merge-clients/03-dropdown.jpeg)

### 5. Selecione "Mesclar Selecionados"

No menu, escolha a opção **"Mesclar Selecionados"**.

![Mesclar selecionados](/images/radiologists/clients/merge-clients/04-merge-selecionados.jpeg)

### 6. Clique em "Executar Ação"

Confirme a operação clicando em **"Executar Ação"**.

![Executar ação](/images/radiologists/clients/merge-clients/05-executar-acao.jpeg)

### 7. Selecione a conta que deseja manter

Escolha qual das contas será a conta principal após a mesclagem.

:::warning
A conta selecionada será mantida. **Todos os dados das demais contas serão transferidos para ela**, e os registros duplicados serão descartados. **Esta ação é irreversível.**
:::

![Selecionar conta principal](/images/radiologists/clients/merge-clients/06-select-account.jpeg)

### 8. Clique em "Mesclar Contas"

![Mesclar contas](/images/radiologists/clients/merge-clients/07-merge-accounts.jpeg)

### 9. Clique em "Continuar" para confirmar

![Confirmar mesclagem](/images/radiologists/clients/merge-clients/08-continue.jpeg)
