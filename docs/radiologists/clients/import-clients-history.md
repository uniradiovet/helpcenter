# Como Visualizar as Importações Anteriores de Clientes

Aprenda como acessar o histórico de importações de clientes na plataforma UniRadio para verificar uploads anteriores e gerenciar os registros da sua clínica.

## Passo a Passo

### 1. Acesse a plataforma UniRadio

Faça login em sua conta e navegue até [app.uniradio.vet.br/exams](https://app.uniradio.vet.br/exams)

### 2. Acesse a seção de Clientes

Clique em **"Clientes"** no menu de navegação para acessar a lista de contas dos seus clientes.

![Seção Clientes](/images/radiologists/clients/import-clients-history/01-clientes.jpeg)

### 3. Abra o menu de opções

Clique no botão de opções (dropdown) para exibir as ações disponíveis para a listagem de clientes.

![Toggle Dropdown](/images/radiologists/clients/import-clients-history/02-dropdown.jpeg)

### 4. Clique em "Importar Clientes"

Selecione a opção **"Importar Clientes"** no menu para abrir o painel de importação.

![Importar Clientes](/images/radiologists/clients/import-clients-history/03-importar-clientes.jpeg)

### 5. Clique em "Ver importações anteriores"

Clique no link **"Ver importações anteriores"** para acessar o histórico de importações já realizadas.

![Ver importações anteriores](/images/radiologists/clients/import-clients-history/04-ver-anteriores.jpeg)

### 6. Visualize a lista de importações

A lista exibirá todas as importações anteriores com informações como data, status e quantidade de registros processados, permitindo acompanhar e verificar cada upload realizado.

![Lista de importações](/images/radiologists/clients/import-clients-history/05-lista-importacoes.jpeg)
