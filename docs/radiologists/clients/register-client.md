# Como Cadastrar um Novo Cliente

Aprenda como adicionar um novo cliente à sua conta na plataforma UniRadio e enviar o convite de acesso.

## Passo a Passo

### 1. Acesse a plataforma UniRadio

Faça login em sua conta e navegue até [app.uniradio.vet.br/exams](https://app.uniradio.vet.br/exams)

### 2. Acesse "Clientes"

No menu lateral, clique em **"Clientes"** para acessar a listagem de clientes cadastrados.

![Clientes](/images/radiologists/clients/register-client/01-clients.jpeg)

### 3. Clique em "Novo Cliente"

Clique no botão **"Novo Cliente"** para iniciar o cadastro.

![Novo Cliente](/images/radiologists/clients/register-client/02-new-client.jpeg)

### 4. Preencha os campos obrigatórios

Informe os dados do cliente, como nome e e-mail, para enviar o convite de acesso.

:::tip Usuário novo
Se o e-mail informado ainda não estiver cadastrado na plataforma, o usuário será criado automaticamente ao aceitar o convite.
:::

:::warning Pular envio de e-mail
Se a opção **"Pular envio de e-mail de convite"** for marcada, o usuário convidado **não receberá nenhuma notificação** por e-mail e será ativado automaticamente.
:::

![Preencher campos](/images/radiologists/clients/register-client/03-fill-fields.jpeg)

### 5. Clique em "Enviar Convite"

Após preencher os dados, clique em **"Enviar Convite"** para concluir o cadastro e notificar o cliente.

![Enviar Convite](/images/radiologists/clients/register-client/04-send-invite.jpeg)

## Descrição dos campos

Para entender o significado de cada campo deste formulário, consulte [Campos para adicionar novo cliente (clínica)](/radiologists/fields/fields-new-client).

Confira nossas [Perguntas Frequentes](/perguntas-frequentes) ou entre em contato com o suporte se encontrar problemas ao cadastrar clientes.
