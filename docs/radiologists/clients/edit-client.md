# Como Editar as Informações de um Cliente

Aprenda como acessar e atualizar os dados cadastrais de um cliente na plataforma UniRadio.

## Passo a Passo

### 1. Acesse a plataforma UniRadio

Faça login em sua conta e navegue até [app.uniradio.vet.br/exams](https://app.uniradio.vet.br/exams)

### 2. Acesse "Clientes"

No menu lateral, clique em **"Clientes"** para visualizar a lista de clientes cadastrados.

![Clientes](/images/radiologists/clients/edit-client/01-clients.jpeg)

### 3. Selecione a clínica

Clique sobre o cliente que deseja editar para abrir as informações da conta.

![Selecionar clínica](/images/radiologists/clients/edit-client/02-select-clinic.jpeg)

### 4. Edite os campos desejados

Atualize as informações que deseja alterar, como nome, e-mail, telefone ou logotipo da clínica.

![Editar campos](/images/radiologists/clients/edit-client/03-edit-fields.jpeg)

### 5. Clique em "Salvar"

Após realizar as alterações, clique em **"Salvar"** para confirmar as mudanças.

![Salvar](/images/radiologists/clients/edit-client/04-save.jpeg)

## Descrição dos campos

Para entender o significado de cada campo desta tela, consulte [Campos para adicionar novo cliente (clínica)](/radiologists/fields/fields-new-client).
