# Como Reativar a Conta de um Cliente

Aprenda como restaurar o acesso de uma conta de cliente desativada na plataforma UniRadio.

## Passo a Passo

### 1. Acesse a plataforma UniRadio

Faça login em sua conta e navegue até [app.uniradio.vet.br/exams](https://app.uniradio.vet.br/exams)

### 2. Acesse "Clientes"

No menu lateral, clique em **"Clientes"** para visualizar a lista de clientes cadastrados.

![Clientes](/images/radiologists/clients/reactivate-client/01-clients.jpeg)

### 3. Clique na aba "Inativos"

Clique na aba **"Inativos"** para listar apenas as contas desativadas.

![Inativos](/images/radiologists/clients/reactivate-client/02-inativos.jpeg)

### 4. Selecione o cliente

Clique sobre o cliente cuja conta deseja reativar.

![Selecionar cliente](/images/radiologists/clients/reactivate-client/03-select-client.jpeg)

### 5. Clique em "Ações"

Clique no botão **"Ações"** para abrir o menu de opções disponíveis para o cliente.

![Menu Ações](/images/radiologists/clients/reactivate-client/04-acoes.jpeg)

### 6. Clique em "Ativar Conta"

Selecione a opção **"Ativar Conta"** no menu de ações.

![Ativar Conta](/images/radiologists/clients/reactivate-client/05-activate-account.jpeg)

### 7. Clique em "Continuar" para confirmar

Clique em **"Continuar"** para confirmar a reativação. O cliente voltará a ter acesso à plataforma imediatamente.

![Confirmar reativação](/images/radiologists/clients/reactivate-client/06-continue.jpeg)
