# Como Editar um Exame

Aprenda como atualizar e modificar os dados de um exame existente na plataforma UniRadio.

## Passo a Passo

### 1. Acesse a plataforma UniRadio

Faça login em sua conta e navegue até [app.uniradio.vet.br/exams](https://app.uniradio.vet.br/exams)

### 2. Selecione o exame desejado

Na lista de exames, clique sobre o exame que deseja editar.

![Selecionar exame](/images/radiologists/exam/exam-edit/01-select-exam.jpeg)

### 3. Clique em "Editar Exame"

![Editar exame](/images/radiologists/exam/exam-edit/02-edit-exam.jpeg)

### 4. Edite os campos desejados

Atualize as informações que precisam ser corrigidas ou complementadas, como tipo de exame, regiões, dados do paciente ou histórico clínico.

![Editar campos](/images/radiologists/exam/exam-edit/03-edit-fields.jpeg)

### 5. Clique em "Salvar" para confirmar as alterações

![Salvar](/images/radiologists/exam/exam-edit/04-save.jpeg)

## Descrição dos campos

Para entender o significado de cada campo deste formulário, consulte [Campos da criação de exame](/radiologists/exam/exam-fields).

Confira nossas [Perguntas Frequentes](/faq) ou entre em contato com o suporte se encontrar problemas ao editar um exame.
