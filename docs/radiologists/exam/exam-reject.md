# Como Rejeitar um Exame

Aprenda como rejeitar um exame na plataforma UniRadio diretamente pelo menu de ações.

## Passo a Passo

### 1. Acesse a plataforma UniRadio

Faça login em sua conta e navegue até [app.uniradio.vet.br/exams](https://app.uniradio.vet.br/exams)

### 2. Clique em "Em Andamento"

Filtre os exames pelo status **"Em Andamento"** para localizar o exame que deseja rejeitar.

![Em Andamento](/images/radiologists/exam/exam-reject/01-in-progress.jpeg)

### 3. Selecione o exame desejado

Clique sobre o exame que deseja rejeitar.

![Selecionar exame](/images/radiologists/exam/exam-reject/02-select-exam.jpeg)

### 4. Clique em "Ações"

Abra o menu de ações disponíveis para o exame.

![Menu de ações](/images/radiologists/exam/exam-reject/03-acoes.jpeg)

### 5. Clique em "Rejeitar"

Selecione a opção **"Rejeitar"** no menu de ações.

![Rejeitar exame](/images/radiologists/exam/exam-reject/04-reject.jpeg)

### 6. Clique em "Continuar" para confirmar

![Confirmar rejeição](/images/radiologists/exam/exam-reject/05-continue.jpeg)
