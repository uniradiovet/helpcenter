# Como Criar um Exame via Servidor PACS

Aprenda como criar e enviar um novo exame na plataforma UniRadio utilizando o servidor PACS para sincronizar os arquivos de imagem.

## Passo a Passo

### 1. Acesse a plataforma UniRadio

Faça login em sua conta e navegue até [app.uniradio.vet.br/exams](https://app.uniradio.vet.br/exams)

### 2. Clique em "Novo Exame"

![Novo Exame](/images/radiologists/exam/pacs-exam-new/01-new-exam.jpeg)

### 3. Clique em "Servidor PACS"

Selecione a opção de envio via servidor PACS.

![Servidor PACS](/images/radiologists/exam/pacs-exam-new/02-server-pacs.jpeg)

### 4. Clique em "Escolher"

Selecione o exame disponível no servidor PACS que deseja importar.

![Escolher exame PACS](/images/radiologists/exam/pacs-exam-new/03-escolher.jpeg)

### 5. Preencha as informações do exame

Preencha todos os campos obrigatórios do formulário, como tipo de exame, regiões, dados do paciente e histórico clínico.

![Informações do exame](/images/radiologists/exam/pacs-exam-new/04-info.jpeg)

### 6. Clique em "Salvar" para enviar o exame

![Salvar](/images/radiologists/exam/pacs-exam-new/05-save.jpeg)

:::tip Dica
Após salvar, o exame será criado como rascunho. Abra-o em uma nova aba para acompanhar o progresso da sincronização com o servidor PACS.
:::

## Descrição dos campos

Para entender o significado de cada campo deste formulário, consulte [Campos da criação de exame](/radiologists/exam/exam-fields).
