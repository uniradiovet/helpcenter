# Como Editar um Modelo de Laudo

Aprenda como acessar e modificar seus modelos de laudos na plataforma UniRadio, mantendo sua documentação sempre atualizada e precisa.

## Passo a Passo

### 1. Acesse a plataforma UniRadio

Faça login em sua conta e navegue até [app.uniradio.vet.br/exams](https://app.uniradio.vet.br/exams)

### 2. Abra o menu lateral

Clique no ícone do menu lateral para expandir as opções de navegação.

![Menu lateral](/images/radiologists/report/edit-report-template/01-menu.jpeg)

### 3. Acesse "Modelos de Laudos"

No menu lateral, clique em **"Modelos de Laudos"** para visualizar a lista de modelos cadastrados.

![Modelos de Laudos](/images/radiologists/report/edit-report-template/02-templates-reports.jpeg)

### 4. Clique em "Editar"

Localize o modelo que deseja modificar e clique no botão **"Editar"**.

![Editar modelo](/images/radiologists/report/edit-report-template/03-edit.jpeg)

### 5. Edite as informações desejadas

Atualize os campos que deseja alterar no formulário do modelo de laudo.

![Editar informações](/images/radiologists/report/edit-report-template/04-edit-info.jpeg)

### 6. Clique em "Salvar"

Após realizar as alterações, clique em **"Salvar"** para confirmar as mudanças no modelo.

![Salvar](/images/radiologists/report/edit-report-template/05-save.jpeg)

## Precisa de Ajuda?

Confira nossas [Perguntas Frequentes](/perguntas-frequentes) ou entre em contato com o suporte se encontrar problemas ao editar modelos de laudos.
