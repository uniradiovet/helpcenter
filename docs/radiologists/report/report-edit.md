# Como Editar um Laudo

Aprenda como acessar, modificar e liberar um laudo na plataforma UniRadio.

## Passo a Passo

### 1. Acesse a plataforma UniRadio

Navegue até [app.uniradio.vet.br/exams](https://app.uniradio.vet.br/exams)

### 2. Selecione os exames com status "Finalizado"

Clique na aba "Finalizado" para listar apenas os exames que já possuem laudo disponível.

![Aba Finalizado](/images/radiologists/report/report-edit/01-finalizado.jpeg)

### 3. Selecione o exame

Clique sobre o exame cujo laudo deseja editar.

![Selecionar exame](/images/radiologists/report/report-edit/02-select-exam.jpeg)

### 4. Clique em "Editar Laudo"

![Editar Laudo](/images/radiologists/report/report-edit/03-edit-report.jpeg)

### 5. Edite as informações do laudo

Realize as alterações desejadas no conteúdo do laudo.

![Editor de laudo](/images/radiologists/report/report-edit/04-editor.jpeg)

### 6. Clique em "Salvar e enviar"

Após concluir as edições, clique em **"Salvar e enviar"** para preparar o laudo para liberação.

![Salvar e enviar](/images/radiologists/report/report-edit/05-save-send.jpeg)

### 7. Clique em "Sim, salvar e liberar" para confirmar

![Confirmar liberação](/images/radiologists/report/report-edit/06-confirm.jpeg)

## Descrição dos campos

Para entender o significado de cada campo do editor de laudo, consulte [Campos do laudo](/radiologists/fields/fields-report).
