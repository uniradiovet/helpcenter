# Como Excluir um Webhook

Aprenda como remover um webhook da sua conta na plataforma UniRadio.

## Passo a Passo

### 1. Acesse a plataforma UniRadio

Faça login em sua conta e navegue até [app.uniradio.vet.br/exams](https://app.uniradio.vet.br/exams)

### 2. Acesse o menu de configurações

No menu lateral, clique no ícone de configurações para abrir as opções da conta.

![Menu de configurações](/images/radiologists/integrations/webhooks-delete/01-menu.jpeg)

### 3. Clique em "Integrações"

No menu de configurações, clique em **"Integrações"** para acessar as opções de integração.

![Integrações](/images/radiologists/integrations/webhooks-delete/02-integrations.jpeg)

### 4. Clique em "Webhooks"

Dentro das integrações, clique em **"Webhooks"** para visualizar os webhooks cadastrados.

![Webhooks](/images/radiologists/integrations/webhooks-delete/03-webhooks.jpeg)

### 5. Clique no ícone de lixeira para excluir o webhook desejado

![Excluir webhook](/images/radiologists/integrations/webhooks-delete/04-delete.jpeg)

### 6. Clique em "Continuar" para confirmar a exclusão

![Confirmar exclusão](/images/radiologists/integrations/webhooks-delete/05-confirm.jpeg)

## Precisa de Ajuda?

Confira nossas [Perguntas Frequentes](/perguntas-frequentes) ou entre em contato com o suporte se encontrar problemas ao excluir webhooks.
