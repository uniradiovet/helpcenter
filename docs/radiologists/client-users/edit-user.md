# Como Editar um Usuário da Conta de um Cliente

Aprenda como atualizar as informações e permissões de um usuário vinculado à conta de um cliente na plataforma UniRadio.

## Passo a Passo

### 1. Acesse a plataforma UniRadio

Faça login em sua conta e navegue até [app.uniradio.vet.br/exams](https://app.uniradio.vet.br/exams)

### 2. Acesse "Clientes"

No menu lateral, clique em **"Clientes"** para visualizar a lista de clientes cadastrados.

![Clientes](/images/radiologists/client-users/edit-user/01-clients.jpeg)

### 3. Selecione a clínica

Clique sobre a clínica para abrir a página de gerenciamento.

![Selecionar clínica](/images/radiologists/client-users/edit-user/02-select-clinic.jpeg)

### 4. Acesse "Usuários da Conta"

Dentro do perfil do cliente, clique em **"Usuários da Conta"** para visualizar os usuários vinculados a ele.

![Usuários da Conta](/images/radiologists/client-users/edit-user/03-users-account.jpeg)

### 5. Selecione o usuário

Clique sobre o nome do usuário que deseja editar.

![Selecionar usuário](/images/radiologists/client-users/edit-user/04-select-user.jpeg)

### 6. Atualize as informações

Edite os campos desejados, como nome, função ou permissões do usuário.

![Atualizar informações](/images/radiologists/client-users/edit-user/05-update-info.jpeg)

### 7. Clique em "Salvar"

Após realizar as alterações, clique em **"Salvar"** para confirmar as mudanças.

![Salvar](/images/radiologists/client-users/edit-user/06-save.jpeg)
