# Como Desativar um Usuário da Conta de um Cliente

Aprenda como desativar o acesso de um usuário vinculado à conta de um cliente na plataforma UniRadio.

## Passo a Passo

### 1. Acesse a plataforma UniRadio

Faça login em sua conta e navegue até [app.uniradio.vet.br/exams](https://app.uniradio.vet.br/exams)

### 2. Acesse "Clientes"

No menu lateral, clique em **"Clientes"** para visualizar a lista de clientes cadastrados.

![Clientes](/images/radiologists/client-users/deactivate-user/01-clients.jpeg)

### 3. Selecione a clínica

Clique sobre a clínica para abrir a página de gerenciamento.

![Selecionar clínica](/images/radiologists/client-users/deactivate-user/02-select-clinic.jpeg)

### 4. Acesse "Usuários da Conta"

Dentro do perfil do cliente, clique em **"Usuários da Conta"** para visualizar os usuários vinculados a ele.

![Usuários da Conta](/images/radiologists/client-users/deactivate-user/03-users-account.jpeg)

### 5. Selecione o usuário

Clique sobre o nome do usuário que deseja desativar.

![Selecionar usuário](/images/radiologists/client-users/deactivate-user/04-select-user.jpeg)

### 6. Clique em "Ações"

Na página do usuário, clique no botão **"Ações"** para exibir as opções disponíveis.

![Ações](/images/radiologists/client-users/deactivate-user/05-acoes.jpeg)

### 7. Clique em "Desativar usuário"

No menu de ações, selecione **"Desativar usuário"** para iniciar o processo de desativação.

![Desativar usuário](/images/radiologists/client-users/deactivate-user/06-deactivate-user.jpeg)

### 8. Clique em "Continuar"

Na janela de confirmação, clique em **"Continuar"** para confirmar a desativação do usuário.

![Continuar](/images/radiologists/client-users/deactivate-user/07-continue.jpeg)

## Precisa de Ajuda?

Confira nossas [Perguntas Frequentes](/perguntas-frequentes) ou entre em contato com o suporte se encontrar problemas ao desativar usuários.
