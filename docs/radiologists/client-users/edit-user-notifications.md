# Como Editar as Notificações de um Usuário da Conta de um Cliente

Aprenda como acessar e configurar as preferências de notificação de um usuário vinculado à conta de um cliente na plataforma UniRadio.

## Passo a Passo

### 1. Acesse a plataforma UniRadio

Faça login em sua conta e navegue até [app.uniradio.vet.br/exams](https://app.uniradio.vet.br/exams)

### 2. Acesse "Clientes"

No menu lateral, clique em **"Clientes"** para visualizar a lista de clientes cadastrados.

![Clientes](/images/radiologists/client-users/edit-user-notifications/01-clients.jpeg)

### 3. Selecione a clínica

Clique sobre a clínica para abrir a página de gerenciamento.

![Selecionar clínica](/images/radiologists/client-users/edit-user-notifications/02-select-clinic.jpeg)

### 4. Acesse "Usuários da Conta"

Dentro do perfil do cliente, clique em **"Usuários da Conta"** para visualizar os usuários vinculados a ele.

![Usuários da Conta](/images/radiologists/client-users/edit-user-notifications/03-users-account.jpeg)

### 5. Selecione o usuário

Clique sobre o nome do usuário cujas notificações deseja configurar.

![Selecionar usuário](/images/radiologists/client-users/edit-user-notifications/04-select-user.jpeg)

### 6. Clique em "Notificações"

Na página do usuário, clique na aba **"Notificações"** para acessar as preferências de alertas.

![Notificações](/images/radiologists/client-users/edit-user-notifications/05-notifications.jpeg)

### 7. Selecione as preferências de notificação

Marque ou desmarque as opções de notificação da conta do cliente conforme desejado, como alertas de rejeição de exames e outras atividades da conta.

![Preferências de notificação](/images/radiologists/client-users/edit-user-notifications/06-preferencias.jpeg)

### 8. Clique em "Salvar"

Após ajustar as configurações de notificação desejadas, clique em **"Salvar"** para confirmar as alterações.

![Salvar](/images/radiologists/client-users/edit-user-notifications/07-save.jpeg)

## Descrição dos campos

Para entender o significado de cada opção de notificação, consulte [Preferências de notificações (Clínica)](/clients/fields/fields-notifications).
