# Como Aceitar Convites de Usuários em Massa

Aprenda como aceitar convites pendentes de múltiplos usuários de uma só vez na plataforma UniRadio.

## Passo a Passo

### 1. Acesse a plataforma UniRadio

Faça login em sua conta e navegue até [app.uniradio.vet.br/exams](https://app.uniradio.vet.br/exams)

### 2. Acesse a lista de clientes

No menu lateral, clique em **"Clientes"** para visualizar as clínicas cadastradas.

![Clientes](/images/radiologists/client-users/bulk-accept-invitation/01-clients.jpeg)

### 3. Selecione a clínica

Clique na clínica cujos usuários deseja gerenciar.

![Selecionar clínica](/images/radiologists/client-users/bulk-accept-invitation/02-select-clinic.jpeg)

### 4. Acesse os usuários da conta

Dentro da clínica, clique na aba de usuários para visualizar os usuários cadastrados.

![Usuários da conta](/images/radiologists/client-users/bulk-accept-invitation/03-users-account.jpeg)

### 5. Selecione os usuários desejados

Marque as caixas de seleção dos usuários cujos convites deseja aceitar.

![Selecionar usuários](/images/radiologists/client-users/bulk-accept-invitation/04-select-users.jpeg)

### 6. Clique no dropdown de ações

Com os usuários selecionados, clique no menu suspenso de ações em massa.

![Dropdown de ações](/images/radiologists/client-users/bulk-accept-invitation/05-dropdown.jpeg)

### 7. Selecione "Aceitar Convite Selecionados"

No menu, escolha a opção **"Aceitar Convite Selecionados"**.

![Aceitar convite](/images/radiologists/client-users/bulk-accept-invitation/06-accept-invite.jpeg)

### 8. Clique em "Executar Ação"

Confirme a operação clicando em **"Executar Ação"** para aceitar os convites dos usuários selecionados.

![Executar ação](/images/radiologists/client-users/bulk-accept-invitation/07-executar-acao.jpeg)
