# Como Obter o Link de Redefinição de Senha de um Usuário

Aprenda como gerar e copiar o link de redefinição de senha de um usuário vinculado à conta de um cliente na plataforma UniRadio.

## Passo a Passo

### 1. Acesse a plataforma UniRadio

Faça login em sua conta e navegue até [app.uniradio.vet.br/exams](https://app.uniradio.vet.br/exams)

### 2. Acesse "Clientes"

No menu lateral, clique em **"Clientes"** para visualizar a lista de clientes cadastrados.

![Clientes](/images/radiologists/client-users/user-password-reset-link/01-clients.jpeg)

### 3. Selecione a clínica

Clique sobre a clínica para abrir a página de gerenciamento.

![Selecionar clínica](/images/radiologists/client-users/user-password-reset-link/02-select-clinic.jpeg)

### 4. Acesse "Usuários da Conta"

Dentro do perfil do cliente, clique em **"Usuários da Conta"** para visualizar os usuários vinculados a ele.

![Usuários da Conta](/images/radiologists/client-users/user-password-reset-link/03-users-account.jpeg)

### 5. Selecione o usuário

Clique sobre o nome do usuário para o qual deseja gerar o link de redefinição de senha.

![Selecionar usuário](/images/radiologists/client-users/user-password-reset-link/04-select-user.jpeg)

### 6. Clique em "Ações"

Na página do usuário, clique no botão **"Ações"** para exibir as opções disponíveis.

![Ações](/images/radiologists/client-users/user-password-reset-link/05-acoes.jpeg)

### 7. Clique em "Link de Redefinição de Senha"

No menu de ações, selecione **"Link de Redefinição de Senha"** para gerar o link de redefinição.

![Link de Redefinição de Senha](/images/radiologists/client-users/user-password-reset-link/06-link-reset-password.jpeg)

### 8. Clique em "Copiar Link"

Clique em **"Copiar Link"** para copiar o link e enviá-lo ao usuário.

![Copiar Link](/images/radiologists/client-users/user-password-reset-link/07-copiar-link.jpeg)

## Precisa de Ajuda?

Confira nossas [Perguntas Frequentes](/perguntas-frequentes) ou entre em contato com o suporte se encontrar problemas ao gerar links de redefinição de senha.
