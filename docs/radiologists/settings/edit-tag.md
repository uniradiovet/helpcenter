# Como Editar uma Tag

Aprenda como alterar o nome de uma tag existente na plataforma UniRadio.

## Passo a Passo

### 1. Acesse a plataforma UniRadio

Faça login em sua conta e navegue até [app.uniradio.vet.br/exams](https://app.uniradio.vet.br/exams)

### 2. Acesse o menu de configurações

No menu lateral, clique no ícone de configurações para abrir as opções da conta.

![Menu de configurações](/images/radiologists/settings/edit-tag/01-menu.jpeg)

### 3. Clique em "Tags"

No menu de configurações, clique em **"Tags"** para visualizar as tags cadastradas.

![Tags](/images/radiologists/settings/edit-tag/02-tags.jpeg)

### 4. Clique em "Editar"

Ao lado da tag que deseja modificar, clique em **"Editar"**.

![Editar](/images/radiologists/settings/edit-tag/03-edit.jpeg)

### 5. Edite o nome da tag

Altere o nome da tag conforme desejado.

:::warning Atenção
Ao editar uma tag, não é possível adicionar outras tags separando por vírgula. Para criar múltiplas tags de uma vez, siga esse tutorial [Criar Tags](/radiologists/settings/create-tag)
:::

![Editar tag](/images/radiologists/settings/edit-tag/04-edit-tag.jpeg)

### 6. Clique em "Salvar"

Após realizar a alteração, clique em **"Salvar"** para confirmar.

![Salvar](/images/radiologists/settings/edit-tag/05-save.jpeg)

## Precisa de Ajuda?
