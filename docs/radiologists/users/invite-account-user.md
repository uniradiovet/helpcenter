# Como Convidar um Novo Usuário para a Conta

Aprenda como conceder acesso à sua equipe enviando convites pela área de configurações da plataforma UniRadio.

## Passo a Passo

### 1. Acesse a plataforma UniRadio

Faça login em sua conta e navegue até [app.uniradio.vet.br/exams](https://app.uniradio.vet.br/exams)

### 2. Acesse o menu de configurações

No menu lateral, clique no ícone de configurações para abrir as opções da conta.

![Menu de configurações](/images/radiologists/users/invite-account-user/01-menu.jpeg)

### 3. Clique em "Usuários"

No menu de configurações, clique em **"Usuários"** para gerenciar os membros da sua conta.

![Usuários](/images/radiologists/users/invite-account-user/02-users.jpeg)

### 4. Clique em "Convidar Usuário"

Clique no botão **"Convidar Usuário"** para iniciar o processo de convite.

![Convidar Usuário](/images/radiologists/users/invite-account-user/03-invite-user.jpeg)

### 5. Preencha as informações do usuário

Informe o nome, e-mail e demais dados necessários para o convite.

![Preencher informações](/images/radiologists/users/invite-account-user/04-fill-info.jpeg)

### 6. Clique em "Enviar convite"

Após preencher os dados, clique em **"Enviar convite"** para finalizar e enviar o convite ao novo usuário.

![Enviar convite](/images/radiologists/users/invite-account-user/05-send-invite.jpeg)

## Descrição dos campos

Para entender o significado de cada campo deste formulário, consulte [Convidar usuário para a conta (Radiologista)](/radiologists/fields/fields-invite-user-account).
