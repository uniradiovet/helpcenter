# Como Reenviar Convite para Usuários da Conta em Massa

Aprenda como reenviar convites pendentes para múltiplos usuários da sua própria conta na plataforma UniRadio.

## Passo a Passo

### 1. Acesse a plataforma UniRadio

Faça login em sua conta e navegue até [app.uniradio.vet.br/exams](https://app.uniradio.vet.br/exams)

### 2. Acesse o menu de configurações

No menu lateral, clique no ícone de configurações para abrir as opções da conta.

![Menu de configurações](/images/radiologists/users/bulk-resend-account-invitation/01-menu.jpeg)

### 3. Acesse os usuários da conta

Clique em **"Usuários"** para visualizar os usuários cadastrados na sua conta.

![Usuários](/images/radiologists/users/bulk-resend-account-invitation/02-users.jpeg)

### 4. Selecione os usuários desejados

Marque as caixas de seleção dos usuários para os quais deseja reenviar o convite.

![Selecionar usuários](/images/radiologists/users/bulk-resend-account-invitation/03-select-users.jpeg)

### 5. Clique no dropdown de ações

Com os usuários selecionados, clique no menu suspenso de ações em massa.

![Dropdown de ações](/images/radiologists/users/bulk-resend-account-invitation/04-dropdown.jpeg)

### 6. Selecione "Reenviar Convite Para Usuários Selecionados"

No menu, escolha a opção **"Reenviar Convite Para Usuários Selecionados"**.

![Reenviar convite](/images/radiologists/users/bulk-resend-account-invitation/05-resend-invite.jpeg)

### 7. Clique em "Executar Ação"

Confirme a operação clicando em **"Executar Ação"** para reenviar os convites aos usuários selecionados.

![Executar ação](/images/radiologists/users/bulk-resend-account-invitation/06-executar-acao.jpeg)

## Precisa de Ajuda?

Confira nossas [Perguntas Frequentes](/faq) ou entre em contato com o suporte se encontrar problemas ao reenviar convites em massa.
